## リンク

### 開く場所を指定する

| 値           | 意味                 |
| ----------- | ------------------ |
| `"_blank"`  | 新しいタブ/ウィンドウで開く     |
| `"_self"`   | 現在のウィンドウ（デフォルト）    |
| `"_parent"` | 親のフレームで開く（フレーム使用時） |
| `"_top"`    | 一番上のフレームで開く        |

```js
<a href="https://www.google.com/" target="_blank" />
```
